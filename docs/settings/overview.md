# Settings

Kaizen's settings are accessible from the sidebar under **Settings**. Here's an overview of what you can configure.

## Settings Pages

| Page | Path | Description |
|------|------|-------------|
| [Agents](agents.md) | `/settings/agents` | Configure AI models and prompts for each agent |
| [Soul & Personality](soul.md) | `/settings/soul` | Customize the agent's personality and communication style |
| [Memory](memory.md) | `/settings/memory` | View and edit the persistent user memory |
| [Voice](voice.md) | `/settings/voice` | Configure voice input and transcription |
| Browser | `/settings/browser` | Toggle incognito mode for browser automation |
| Theme | `/settings/theme` | Toggle the Kaizen theme for generated HTML pages and link previews |
| [Backup & Recovery](backup.md) | `/settings/recovery` | Export and import all Kaizen data |

## Theme Settings

### Theme Kit

When enabled, HTML pages generated by Kaizen (dashboards, reports, etc.) use a consistent dark theme with shadcn-styled components. When disabled, the AI model chooses its own styling.

### Link Previews

When enabled, URLs shared in chat are automatically unfurled with a preview card showing the page title and description. Enabled by default.

## Browser Settings

Toggle **Incognito Mode** to launch the browser without cookies, history, or cache. Useful for tasks that require a clean browser state. Takes effect on the next browser launch.
