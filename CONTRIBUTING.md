# Contributing to Kaizen

Thanks for your interest in Kaizen!

## Current Status

Kaizen is in early development. We're **not accepting pull requests** at this time, but this will change as the project matures.

## How You Can Help

- **Bug reports.** Found something broken? [Open an issue](https://github.com/kaizen-labs-ai/kaizen/issues/new) with steps to reproduce.
- **Feature requests.** Have an idea? [Open an issue](https://github.com/kaizen-labs-ai/kaizen/issues/new) and describe what you'd like to see.
- **Questions.** Use [GitHub Discussions](https://github.com/kaizen-labs-ai/kaizen/discussions) for general questions.

## Reporting Security Issues

Please do **not** open a public issue for security vulnerabilities. See [SECURITY.md](./SECURITY.md) for responsible disclosure instructions.
